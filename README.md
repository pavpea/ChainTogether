# ChainTogether 模组介绍

## 模组功能

ChainTogether 是一个 Minecraft Forge 模组，允许玩家通过拴绳互相绑定，形成一条视觉上的链条连接，增强多人游戏的互动体验。

### 核心功能
1. **玩家绑定**：使用拴绳右击其他玩家，可以将两个玩家绑定在一起。
2. **链条渲染**：绑定的玩家之间会显示一条视觉链条，直观展示绑定关系。
3. **共享复活点**：绑定的玩家会共享复活点，确保团队合作的连续性。
4. **绑定状态管理**：支持玩家解绑，解绑后链条会立即消失。
5. **中文提示**：所有操作都有中文提示信息，提升游戏体验。

## 游玩指令

/chain bind <player1> <player2> x y z - 绑定两个玩家

/chain unbind <player> - 解除玩家绑定

/chain length <multiplier> - 设置粒子线长度倍数

/chain reload - 重新加载配置

/chain list - 列出所有被绑定的玩家

/chain transparency <10~100>  - 设置粒子线透明度

### 绑定玩家

1. **获取拴绳**：在游戏中获取至少一根拴绳（Lead）。
2. **绑定操作**：手持拴绳，右击目标玩家。
3. **绑定提示**：绑定成功后，双方都会收到"已成功与 [玩家名] 绑定！"的提示。
4. **消耗物品**：非创造模式下，绑定操作会消耗一根拴绳。

### 解绑玩家

1. **自然解绑**：当绑定的玩家距离过远时，链条会自动断开。
2. **解绑提示**：解绑成功后，双方都会收到"已成功与 [玩家名] 解绑！"的提示。

### 注意事项

1. **绑定限制**：一个玩家只能同时绑定一个其他玩家。
2. **距离限制**：绑定的玩家之间有最大距离限制，超过限制会自动解绑。
3. **视觉效果**：链条的渲染效果会根据玩家之间的距离和角度自动调整。
4. **兼容性**：模组兼容 Minecraft 1.20.1 & 1.21.1 版本的 Forge 服务器。

## 技术特点

1. **网络同步**：使用 Forge 网络包系统，确保客户端和服务器的绑定状态同步。
2. **高效渲染**：优化的渲染逻辑，只渲染真正绑定的玩家对之间的链条。
3. **多语言支持**：内置中文和英文语言支持，可根据游戏语言自动切换。
4. **配置灵活**：通过配置文件可以调整链条长度等参数。

ChainTogether 模组为多人游戏增添了新的互动方式，让玩家之间的合作更加紧密和有趣。无论是探索未知的世界，还是挑战危险的地下城，绑定在一起的玩家都能感受到更强的团队凝聚力。
